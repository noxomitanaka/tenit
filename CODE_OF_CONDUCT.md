# Contributor Covenant Code of Conduct

## 私たちの誓約

私たちメンバー、コントリビューター、リーダーは、年齢、体格、目に見える/目に見えない障害、民族性、性別、性同一性、経験レベル、教育、社会経済的地位、国籍、外見、人種、宗教、性的指向にかかわらず、ハラスメントのないコミュニティとして参加することを誓約します。

オープンで歓迎的、多様で、包括的で健全なコミュニティに貢献するために行動します。

## 私たちの基準

ポジティブな環境作りに貢献する行動:

- 他者への共感と親切
- 異なる意見・視点・経験への敬意
- 建設的なフィードバックを与え、優雅に受け取る
- ミスから学び、影響を受けた人に謝罪する
- 個人ではなくコミュニティ全体にとって最善のものに焦点を当てる

容認されない行動:

- 性的な言葉や画像、性的な注目や接近
- トローリング、侮辱的・軽蔑的なコメント、個人的・政治的攻撃
- 公的または私的な嫌がらせ
- メールアドレス等の物理的・電子的住所の許可なき公開
- 業務環境において妥当でないとみなされる他の行為

## 適用範囲

GitHub Issues / Pull Requests / Discussions / 公式コミュニティスペース全般。Tenit の公式代表として行動するすべての場面。

## 違反の報告

容認されない行動を観察した場合、プロジェクトメンテナへ GitHub プロフィール経由で連絡してください。すべての報告は迅速かつ公正に審査・調査されます。

報告者のプライバシーとセキュリティは尊重されます。

## ガイドライン

メンテナは違反者に対し以下のいずれかを適用します。

1. **訂正** — プライベートな注意
2. **警告** — 一定期間の行動制限
3. **一時的禁止** — コミュニティへのアクセス禁止（期間付き）
4. **永久追放** — 恒久的な参加禁止
